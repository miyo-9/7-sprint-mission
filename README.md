# 🎯 스프린트 미션 프로젝트 (Node.js 백엔드 7기)

이 프로젝트는 Codeit Node.js 백엔드 스프린트 미션으로,  
**클래스 기반 설계와 서비스 로직 구현**을 연습하기 위해 만들어졌습니다.

## 📌 프로젝트 목적

- JavaScript 클래스(Class) 문법 이해
- 상속(Inheritance) 구조 경험
- Article / Product 도메인 모델 구현
- Service 패턴으로 데이터 처리 흐름 구성
- 객체 생성, 데이터 저장·조회·수정 기능 연습
- GitHub PR 및 코드 리뷰 경험 쌓기

## 📂 구현 기능

### 📝 Article 기능
- 게시글 생성
- 게시글 목록 조회
- 게시글 좋아요 증가
- ArticleService를 통한 중앙 관리

### 🛒 Product 기능
- 상품 생성
- 상품 목록 조회
- 관심상품(favorite) 증가
- ProductService로 관리
- ElectronicProduct(상속) 구현

## 🛠 기술 스택
- Node.js
- JavaScript (ES Module)
- Git / GitHub PR
